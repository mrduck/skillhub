SkillHub 隐私政策

最后更新：2026 年 6 月 12 日

数据收集

SkillHub 是一个纯客户端 Chrome 扩展，不收集、不存储、不上传任何个人数据到我们的服务器。我们没有后端服务器。

本地存储的数据

以下数据仅存储在您的浏览器本地，不会传输到任何第三方：

  • 收藏的 Skill：您收藏的 Skill ID 列表，存储在 Chrome Storage 中，用于同步到同一 Google 账号的其他设备
  • Skill 目录数据：从公开数据源（GitHub、skills.sh）获取的 Skill 元数据，缓存在 IndexedDB 中

第三方服务

SkillHub 使用以下第三方服务，请注意它们的隐私政策：

  • ExtensionPay：处理 Pro 付费。支付信息由 ExtensionPay 直接处理，我们无法访问您的支付详情。详见 ExtensionPay 隐私政策
  • GitHub API：获取 Skill 仓库的元数据（star 数、描述等）。不会发送任何用户数据

权限说明

  • storage：存储收藏列表和 Skill 数据缓存
  • alarms：定时更新 Skill 数据
  • activeTab：在 GitHub 页面右键一键安装 Skill
  • scripting：在 GitHub 页面注入安装脚本
  • downloads：导出收藏数据
  • notifications：数据更新通知
  • contextMenus：右键菜单快捷操作

数据删除

您可以随时在扩展设置中导出或清除数据。卸载扩展将自动清除所有本地数据。如需删除 ExtensionPay 账号数据，请联系 ExtensionPay 支持。
